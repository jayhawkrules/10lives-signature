How to add it in Gmail

  1. Click Copy signature above.
  2. In Gmail, open Settings (⚙️) → See all settings → General.
  3. Scroll to Signature, create or select one, click into the box.
  4. Paste (⌘V). The logo, icons and formatting come with it.
  5. Scroll down and click Save Changes.
Two signatures? Gmail lets you set a different signature for new emails vs replies/forwards (just below the signature box). Use the Full version for new emails and the Reply (compact) version for replies — switch them with the toggle above the preview.
Other apps: Outlook · Apple Mail · Workspace-wide
Outlook (new / web): Settings → Mail → Compose and reply → Email signature → click into the box → paste (⌘V) → Save.

Apple Mail: Mail → Settings → Signatures → pick the account → create a signature → uncheck “Always match my default message font”, then paste (⌘V).

Whole-company footer: a Workspace admin can apply one in Admin console → Apps → Gmail → Compliance → Append footer — use Copy HTML source and paste it there.

Tip: Test in Gmail opens a compose window so you can paste and email it to yourself to see it render in a real inbox.
